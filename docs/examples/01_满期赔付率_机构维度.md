# 例 1:各机构满期赔付率(剔摩)

## 业务问题
"看下今年到现在为止,各三级机构的剔摩满期赔付率,从高到低排,我想知道哪些机构在亏损边缘。"

## 思考
- **指标**:满期赔付率 = (已决赔款 + 未决赔款) / 满期保费 — 见 `business-rules.md` 四.1
- **时间**:今年至今 → `policy_date` 在 `2025-01-01` 至 `CURRENT_DATE`(假设今年是 2025)
- **维度**:`org_level_3`
- **筛选**:剔摩 → `customer_category != '摩托车'`
- **铁律**:满期保费用 `premium * earned_ratio`;policy 与 claims JOIN 必须先去重批改副本

## SQL

```sql
WITH p AS (
  SELECT
    org_level_3,
    policy_no,
    SUM(premium) AS net_premium,
    SUM(premium * LEAST(1, GREATEST(0,
      date_diff('day', insurance_start_date, LEAST(CURRENT_DATE, insurance_end_date)) / 365.25
    ))) AS earned_premium
  FROM policy_all
  WHERE customer_category != '摩托车'
    AND policy_date >= DATE '2025-01-01'
    AND policy_date <= CURRENT_DATE
  GROUP BY org_level_3, policy_no
),
c AS (
  SELECT policy_no,
         SUM(settled_amount) AS paid,
         SUM(pending_amount) AS outstanding
  FROM claims_all
  GROUP BY policy_no
)
SELECT
  p.org_level_3                                        AS 机构,
  COUNT(DISTINCT p.policy_no)                          AS 保单数,
  ROUND(SUM(p.earned_premium) / 10000, 1)              AS 满期保费_万元,
  ROUND(COALESCE(SUM(c.paid + c.outstanding), 0) / 10000, 1) AS 已发生赔款_万元,
  ROUND(
    COALESCE(SUM(c.paid + c.outstanding), 0)
    / NULLIF(SUM(p.earned_premium), 0) * 100,
  2) AS 满期赔付率_pct
FROM p
LEFT JOIN c USING (policy_no)
GROUP BY p.org_level_3
ORDER BY 满期赔付率_pct DESC NULLS LAST;
```

## 业务解读模板

> "截至今天,机构 X 满期赔付率 94.62%,已逼近成本线,主要拖累在营业出租租赁(占该机构保费 35%)和特种车;
> 公司均值约 65%,天府/宜宾在均值附近;
> 建议优先核查机构 X 的营业出租租赁核保口径,并对赔付率 > 80% 的中小机构启动专项分析。"
