# 例 2:各客户类别出险率(年化)

## 业务问题
"今年各客户类别的出险率怎么样?把摩托车单独算,其他放一起。"

## 思考
- **指标**:出险率 = 出险案件数 / 年化已赚暴露 — 见 `business-rules.md` 四.2
- **关键**:必须**年化**,不能用 `案件数 / 保单数`(被未满期保单稀释)
- **简化口径**:年化暴露 ≈ SUM(满期保费 / 平均保费),实战常用 `SUM(已赚天数) / 365` 作为车年
- **本例采用**:出险率 = SUM(claim_cases) / (SUM(earned_days) / 365),其中 earned_days 是单保单已过保险天数

## SQL

```sql
WITH p AS (
  SELECT
    policy_no,
    customer_category,
    -- 已赚天数(0 ~ 365):不超过保险止期、不超过今天
    GREATEST(0, LEAST(
      date_diff('day', insurance_start_date, LEAST(CURRENT_DATE, insurance_end_date)),
      date_diff('day', insurance_start_date, insurance_end_date)
    )) AS earned_days,
    SUM(claim_cases) AS claim_cases
  FROM policy_all
  WHERE policy_date >= DATE '2025-01-01'
  GROUP BY policy_no, customer_category, insurance_start_date, insurance_end_date
)
SELECT
  customer_category                                   AS 客户类别,
  COUNT(DISTINCT policy_no)                           AS 保单数,
  ROUND(SUM(earned_days) / 365.0, 0)                  AS 年化暴露_车年,
  SUM(claim_cases)                                    AS 出险案件数,
  ROUND(
    SUM(claim_cases) / NULLIF(SUM(earned_days) / 365.0, 0) * 100,
  2) AS 年化出险率_pct
FROM p
GROUP BY customer_category
ORDER BY 年化出险率_pct DESC NULLS LAST;
```

## 预期输出(示意)

| 客户类别 | 保单数 | 年化暴露_车年 | 出险案件数 | 年化出险率_pct |
|---|---|---|---|---|
| 营业货车 | 17,493 | 9,832 | 4,231 | 43.03 |
| 营业出租租赁 | 612 | 387 | 156 | 40.31 |
| 非营业个人客车 | 358,420 | 198,234 | 21,098 | 10.64 |
| 摩托车 | 173,591 | 79,103 | 1,089 | 1.38 |
| ... | ... | ... | ... | ... |

## 业务解读模板

> "营业货车出险率 43%(约 9.3 倍于个人客车),是最大风险源,尤其在 2-9 吨段;
> 摩托车出险率 1.4%,但案均极低;
> 建议:营业货车按吨位分段重新定价,大型货车进一步限承保。"
