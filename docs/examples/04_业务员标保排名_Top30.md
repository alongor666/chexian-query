# 例 4:今年业务员标保 Top30 + 跨机构归属

## 业务问题
"给我列出今年签单保费 Top30 业务员,显示他们归属机构、机构内排名,以及标保增速。"

## 思考
- **指标**:签单保费 SUM(premium) 按 policy_date 归属本年
- **业务员归属规则**(business-rules.md 七):
  - 默认:`dim_salesman.organization`
  - 例外:`organization='未分配'` → 按保单 `org_level_3` 归属
- **维度**:`salesman_name + 归属机构`
- **同比**:本年 vs 去年同期(同 policy_date 月份范围)

## SQL

```sql
WITH cur AS (
  SELECT
    p.salesman_name,
    -- 归属机构:dim 优先,'未分配' 时按保单
    COALESCE(NULLIF(s.organization, '未分配'), p.org_level_3) AS attribution_org,
    SUM(p.premium) AS premium_2025
  FROM policy_all p
  LEFT JOIN dim_salesman s USING (salesman_name)
  WHERE p.policy_date >= DATE '2025-01-01'
    AND p.policy_date < DATE '2026-01-01'
  GROUP BY p.salesman_name, attribution_org
),
prev AS (
  SELECT
    p.salesman_name,
    COALESCE(NULLIF(s.organization, '未分配'), p.org_level_3) AS attribution_org,
    SUM(p.premium) AS premium_2024
  FROM policy_all p
  LEFT JOIN dim_salesman s USING (salesman_name)
  WHERE p.policy_date >= DATE '2024-01-01'
    AND p.policy_date < DATE '2025-01-01'
  GROUP BY p.salesman_name, attribution_org
)
SELECT
  cur.salesman_name                                          AS 业务员,
  cur.attribution_org                                        AS 归属机构,
  ROUND(cur.premium_2025 / 10000, 1)                         AS 今年标保_万元,
  ROUND(COALESCE(prev.premium_2024, 0) / 10000, 1)           AS 去年标保_万元,
  ROUND(
    (cur.premium_2025 - COALESCE(prev.premium_2024, 0))
    / NULLIF(prev.premium_2024, 0) * 100,
  1) AS 同比增速_pct,
  ROW_NUMBER() OVER (ORDER BY cur.premium_2025 DESC)         AS 总排名,
  ROW_NUMBER() OVER (PARTITION BY cur.attribution_org ORDER BY cur.premium_2025 DESC)
                                                             AS 机构内排名
FROM cur
LEFT JOIN prev USING (salesman_name, attribution_org)
QUALIFY 总排名 <= 30
ORDER BY 总排名;
```
