# 例 5:续保率(交商同保口径,按机构)

## 业务问题
"看下今年各机构续保率怎么样,按交商同保口径,排除摩托。"

## 思考
- **续保率口径**(business-rules.md 四.5):
  - 分母:上一年保险起期在统计期内 + `is_commercial_insure='套单'` 的保单
  - 分子:分母中,本年 `renewal_policy_no` 引用了它的保单数
- **筛选**:排除 `customer_category='摩托车'`
- **维度**:`org_level_3`(按上年保单的归属机构)

## SQL

```sql
-- 思路:
-- 1) 找出"应续"集合:上一年(2024)有 套单 + 非摩托 的保单
-- 2) 找出"已续":本年 renewal_policy_no 引用了 应续保单号
-- 3) 续保率 = 已续 / 应续

WITH due_2024 AS (
  -- 上一年(2024)的"应续"保单:套单 + 非摩托
  SELECT
    policy_no,
    org_level_3,
    SUM(premium) AS prev_premium
  FROM policy_all
  WHERE is_commercial_insure = '套单'
    AND customer_category != '摩托车'
    AND insurance_start_date >= DATE '2024-01-01'
    AND insurance_start_date <  DATE '2025-01-01'
  GROUP BY policy_no, org_level_3
),
renewed_in_2025 AS (
  -- 2025 已续保:renewal_policy_no 引用了 due_2024 的保单号
  SELECT DISTINCT renewal_policy_no AS prev_policy_no
  FROM policy_all
  WHERE policy_date >= DATE '2025-01-01'
    AND policy_date <  DATE '2026-01-01'
    AND renewal_policy_no IS NOT NULL
)
SELECT
  d.org_level_3                                              AS 机构,
  COUNT(*)                                                   AS 应续保单,
  COUNT(*) FILTER (WHERE r.prev_policy_no IS NOT NULL)       AS 已续保单,
  ROUND(
    COUNT(*) FILTER (WHERE r.prev_policy_no IS NOT NULL) * 100.0 / COUNT(*),
  2) AS 续保率_pct,
  ROUND(SUM(d.prev_premium) / 10000, 1)                      AS 应续保费_万元
FROM due_2024 d
LEFT JOIN renewed_in_2025 r ON r.prev_policy_no = d.policy_no
GROUP BY d.org_level_3
ORDER BY 续保率_pct DESC;
```

## 业务解读模板

> "整体续保率 73%(同行业平均约 70%),武侯/高新表现最好(>75%),
> 末位三家机构续保率仅 60% 左右,核查原因——是赔付率高导致放弃,还是服务差被竞品挖走。
> 应续保费 X 亿元,潜在流失 Y 亿元。"
